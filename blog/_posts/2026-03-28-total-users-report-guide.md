The total users metric is the first number most Microsoft 365 teams look for when they want a quick view of tenant size. In M365 Tenant Reporter, that number comes from the live user inventory collected through Microsoft Graph after delegated sign-in.

## Why the total user count matters

It gives operations teams a fast baseline before they move into licensed users, unlicensed users, shared mailboxes, or group-heavy investigations. If the baseline suddenly shifts, the rest of the reporting conversation changes with it.

## How to use the metric operationally

- Validate tenant growth before a licensing review.
- Compare the latest snapshot with a previous export.
- Decide whether you need to move from overview cards into the full user report.

## When to export

Export the user report when the overview metric raises a question that needs row-level detail. The HTML or Excel export is often the easiest way to hand that follow-up review to another team.
