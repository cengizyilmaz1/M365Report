Licensed users are often the fastest way to understand where actual service consumption is concentrated. M365 Tenant Reporter normalizes assigned license information so the report stays readable instead of exposing raw Graph payloads.

## What the report highlights

The licensed user view shows which users have one or more assigned SKUs, how many licenses are attached, and which resolved SKU names are present on the account.

## Best use cases

- Audit who currently consumes paid licensing.
- Prepare for procurement conversations.
- Identify heavy-license users before optimization work.

## Recommended export format

Excel is usually the most convenient format for licensed user reviews because finance, operations, and support teams can annotate the workbook without needing a separate transformation step.
